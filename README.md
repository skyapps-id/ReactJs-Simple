# ReactJs Simple Editable

### Stack

- ReactJs 18.2.0
- React Redux 8.0.2
- Firebase Realtime Database

### Manual Installation

1. Install Package

   ```sh
   $ yarn install
   ```

2. Run Project
   ```sh
   $ yarn start
   ```
